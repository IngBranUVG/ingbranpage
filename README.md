# ingbran.tech

Pagina estatica profesional para promocionar servicios de programacion web, bases de datos, infraestructura TI y mantenimiento de computadoras.

## Estructura

- `index.html`: contenido principal, SEO basico y enlaces de contacto.
- `styles.css`: identidad visual, responsive design y animaciones.
- `script.js`: menu movil, efectos de aparicion, contadores y canvas animado.
- `assets/`: imagen principal y favicon.
- `CNAME`: dominio personalizado para GitHub Pages (`ingbran.tech`).
- `robots.txt` y `sitemap.xml`: archivos basicos para SEO.
- `.gitignore`: reglas minimas para mantener limpio el repositorio.

## Abrir localmente

Puedes abrir `index.html` directamente en el navegador. Tambien puedes servirlo con cualquier servidor estatico:

```bash
python -m http.server 8080
```

Luego visita `http://localhost:8080`.

## Publicar

Este proyecto no necesita build. Puedes subir la carpeta completa a GitHub y publicarla con GitHub Pages, Netlify, Vercel o el hosting conectado a `ingbran.tech`.

Para GitHub Pages:

1. Sube estos archivos a la rama que usaras para publicar.
2. En GitHub, entra a `Settings > Pages`.
3. Selecciona la rama y carpeta donde quedo `index.html`.
4. Verifica que el dominio personalizado sea `ingbran.tech`.
5. Configura los DNS del dominio segun las instrucciones de GitHub Pages.
